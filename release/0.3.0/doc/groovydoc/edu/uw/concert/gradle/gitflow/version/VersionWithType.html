[Groovy] Class VersionWithType

    • edu.uw.concert.gradle.gitflow.version.VersionWithType
    • All Implemented Interfaces and Traits:
      Comparable

      class VersionWithType
extends Object

      Combines a com.github.zafarkhaja.semver.Version with a VersionType using the Delegation Pattern. This is needed because inheritance cannot be used due to the package-private constructors of com.github.zafarkhaja.semver.Version. This class therefore delegates to com.github.zafarkhaja.semver.Version.
